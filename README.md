### 📊 [ML for Venture Capital Performance](https://github.com/moshu0517/ml-for-vc-performance)
**Duke Capstone: Identifying drivers of top-performing VC funds**

`NLP` `TF-IDF` `NMF` `Elastic Net` `Python` `Fashion Analytics`
